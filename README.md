# Library-website


📚 World's Most Popular Books Website
A clean and organized web project showcasing 50 of the most popular books globally. This project serves as a digital library where users can explore literary classics and modern bestsellers, view detailed statistics, and read summaries to find their next favorite book.

🚀 Features
Curated Collection: Includes detailed entries for 50 world-renowned books.

Smart Navigation:

Kategoriler (Categories): Browse books filtered by their specific genres.

Yazarlar (Writers): Explore works organized by their creators.

Comprehensive Book Data: Every book entry includes:

📖 Summary: A concise overview of the book's plot or core message.

📄 Page Count: Practical information for readers.

✍️ Author Details: Credits to the writer of the work.

⭐ Reader Ratings: Point rates to help users gauge popularity.

🛠️ Tech Stack
HTML5: The core structure and content organization of the entire site.
